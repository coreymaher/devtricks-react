Imperative

Tell the interpreter what to do


function onClick(e) {
    document.querySelectorAll('#nav .selected').forEach((el) {
        el.classList.remove('selected');
    });

    e.target.classList.add('selected');
}

Declarative

Describe the result you want


function render() {
    return (
        <nav>
            <NavItem className="selected">Foo</NavItem>
            <NavItem>Bar</NavItem>
        </nav>
    );
}

View

React.createElement


React.createElement(type, [props], [...children]);

type - tag name (div, p, ...) or React component

props - optional props/attributes (href, src, ...)

children - optional children react elements

ReactDOM.render


ReactDOM.render(element, container[, callback])

element - root react element

container - DOM element to inject the root react element into

callback - optional callback which is run after the root element is rendered
